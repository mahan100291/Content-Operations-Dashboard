# Content Operations Dashboard

A compact Next.js Pages Router project for a content management and publishing operations app:

- Next.js 15+, React, and TypeScript
- Pages Router today, with `middleware.ts` handling routing concerns that previously lived in Express
- CSS Modules for styling
- Motion for interface animation
- Radix UI and Base UI headless accessible components
- Recharts for data visualization

## Run

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

To exercise the middleware route, open `http://localhost:3000/go/team-growth`.
